Node.js® is a JavaScript runtime built on Chrome's V8 JavaScript engine. Node.js uses an event-driven, non-blocking I/O model that makes it lightweight and efficient. Node.js' package ecosystem, npm, is the largest ecosystem of open source libraries in the world.

Trade-offs

Multithreaded blocking Singlethreaded non-blocking
Scales vertically Scales horizontally
More threads, more active connections, more resources Less resources, connections are cheap
More context switches, more use of CPU caches Intermediate data stays allways on the same CPU. More efficent
Easier to debug Harder to follow code throughout callbacks and different threads

Filtered ls


//load modules
var fs = require('fs')
var path = require('path')

//read arguments
var folder = process.argv[2]
var ext = '.' + process.argv[3]

//callback example
fs.readdir(folder, function (err, files) {
  //error treatment
  if (err) return console.error(err)

  //another callback
  files.forEach(function (file) {
    if (path.extname(file) === ext) {
      console.log(file)
    }
  })
})
